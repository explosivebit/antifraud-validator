# Session Plan: First Lesson

> Duration: ~40 minutes | Format: live coding + demo

## Flow

| Step | Action                              | Duration | Notes |
|------|-------------------------------------|----------|-------|
| 1    | Clone repo, explore starter code    | 5 min    | `git clone` + quick walkthrough |
| 2    | Configure `CLAUDE.md` together      | 5 min    | Explain primacy-recency, rules vs docs |
| 3    | Discuss the task, ask Claude Code   | 5 min    | Show how agent reads CLAUDE.md → docs |
| 4    | Implement night transfer rule       | 10 min   | Live coding with agent assistance |
| 5    | Run tests, verify                   | 5 min    | `node validator.test.js` |
| 6    | Commit + push + open PR             | 5 min    | `git commit` → `git push` → `gh pr create` |
| 7    | Review PR on GitHub, approve/reject | 5 min    | Show GitHub PR interface |

## Offline Replay

Students who missed the session can:
1. Watch the recording (link shared in chat)
2. Clone the same repo
3. Follow steps 1-7 at their own pace
4. Submit their own PR as the homework assignment
